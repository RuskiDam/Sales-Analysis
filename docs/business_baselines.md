# Business Baselines

## Profit Margin Baseline

The profit margin baseline is 50%.

Profit margin movement is measured against the 50% baseline.

Example:
If profit margin is 56.06%, it is 6.06% above baseline, which is a 6.06%
increase.

If profit margin is 47.25%, it is 2.75% below baseline, which is a 2.75%
decrease.

## MoM Revenue Growth Baseline

The MoM revenue growth baseline is 5%.

Above baseline means performance improved.
Below baseline means performance weakened.

When answering baseline questions, include actual value, baseline value,
difference from baseline, and whether that is good or bad.
